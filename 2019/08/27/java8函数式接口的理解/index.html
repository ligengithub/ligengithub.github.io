Java8新特性(一) 函数式接口
Created 2019-08-27| Updated 2019-10-22

1函数式接口

  • 函数式接口在java.util.function包下。
  • 函数式接口中只有一个抽象方法,(静态方法，默认方法可以有多个)
  • 函数式接口可以使用lambda表达式

    有以下大四类

  • Function 函数有参数，有返回值
  • Cosumer 消费者,接收参数，不返回
  • Supplier 生产者，不接受参数有返回
  • Predicate 接收参数，返回值为boolean类型，通常用作对传入参数的校验

fun包下面所有的类都归属于这四种里面。。区别和不同无非就是传入的参数类型
和返回值类型不同罢了
