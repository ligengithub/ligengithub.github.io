Shell的各种运算符

加减乘除取余就不说了，和平常天天用的一样

expr

expr 通常后面跟一个运算表达式,表示一个运算。
如

关系运算
  • -eq 等于（equal）
  • -ne 不等（not equal）
  • -gt 大于（greater than）
  • -lt 小于 （letter than）
  • -ge 大于等于（greater equal）
  • -le 小于等于 （letter equal）
布尔
  • ！ 非
  • -a 与（and）
  • -o 或（or）
逻辑与，或一点没变
  • &&
  • ||
    字符串
  • = 相等
  • ！= 不等
  • -z 长度为0 返回true
  • -n 长度不为0 返回true
  • $ 是否为空，非空返回true
文件相关,很多，这里就写几个常用的，
  • -d 是否是目录
  • -f 普通文件
  • -r 可读
  • -w 可写
  • -x 可执行
  • -s 文件是否为空，大小为0
  • e 文件或目录是否存在
