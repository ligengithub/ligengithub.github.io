shell参数传递(三)
Created 2019-09-23| Updated 2019-09-26

参数传递

参数传入

作用在执行shell脚本的时候，向shell脚本传递参数

参数接收

其它

  • $# 传递到脚本的参数个数
  • $* 传递给脚本的所有参数 “$*” 传入 1 2 3 输出 相当于传入一个参数
  • $$ 脚本运行的当前进程id号
  • $! 后台运行的最后一个进程id号
  • $@ 传递给脚本的所有参数 只在使用双引号与”$*”体现出区别 传入 1 2 3 输出 “1” “2” “3” 相当于传入3个参数
  • $- 显示Shell使用的当前选项
  • $? 显示shell最后的退出状态 0 表示没有错误，其它值表示有错误
